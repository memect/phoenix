- **文档类型**: 北交所年报，结构高度标准化
- **已标注文档**: 3篇（2529252c...、2821be04...、54d78495...）
- **字段数量**: 18个（8个基本信息 + 10个财务数据）

## 提取策略

### 第一阶段：表格结构化提取 ✅

#### 1. 基本信息字段（第二节 公司概况）

**数据来源**：
- 第二节 → 一、基本信息（表格ID 50）
  - 公司中文全称
  - 证券简称
  - 证券代码
  - 法定代表人
  
- 第二节 → 四、企业信息（跨页表格ID 56）
  - 行业分类
  - 主要产品与服务项目
  - 普通股总股本
  - 实际控制人（"实际控制人及其一致行动人"字段）

**提取方法**：
- 遍历 `document.iter_nodes("title")`，定位"第二节"
- 在该节点下找"一、基本信息"和"四、企业信息"子标题
- 提取对应的表格节点，按行名匹配字段

#### 2. 财务数据字段（第三节 会计数据和财务指标）

**数据来源**：
- 第三节 → 一、盈利能力（表格ID 68）
  - 营业收入_2024 / _2023：行名="营业收入"，列名="2024年" / "2023年"
  - 毛利率_2024：行名="毛利率%"，列名="2024年"
  - 归属于上市公司股东的净利润_2024 / _2023：行名="归属于上市公司股东的净利润"（精确匹配）
  - 基本每股收益_2024：行名="基本每股收益"，列名="2024年"

- 第三节 → 二、营运情况（跨页表格ID 71）
  - 资产总计_2024末：行名="资产总计"，列名="2024年末"
  - 负债总计_2024末：行名="负债总计"，列名="2024年末"
  - 归属于上市公司股东的净资产_2024末：行名="归属于上市公司股东的净资产"（精确匹配）
  - 资产负债率_2024末：行名="资产负债率%（合并）"，列名="2024年末"

**提取方法**：
- 定位"第三节" → "一、盈利能力" / "二、营运情况"
- 提取表格，解析表头（列名映射）
- 按行名精确匹配，按列名提取对应年度数据

**关键处理**：
- ✅ 精确匹配行名，避免"归属于上市公司股东的净利润"被"加权平均净资产收益率（依据归属于上市公司股东的净利润计算）"误匹配
